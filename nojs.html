Hello, I'm a web page.

You are only seeing me because a) you are using a very, very, very old browser or b) you are using a script blocker.

Have you ever navigated to a website only to find it unusable because you forgot to enable Javascript? If, like me, you use NoScript (or a similar plugin) because you care about performance and security and being tracked by creepy bots, this probably happens to you on a daily basis. If, unlike me, you enjoy the web in all its Javascript-y glory...

Imagine if 80% of the websites you visited weren't fully functional unless you disabled Javascript. Layout garbled. Dropdown menus unhidden. Carousels a mess of overlapping images and text. Oh, and the worse thing: checkout processes broken, often fatally.

Or, if you're really unlucky, just a message saying "please disable Javascript to view this page".

Graceful degradation, also known more positively as "progressive enhancement"

Use it.

And for the love of the web, no more Javascript dropdown menus.
